Aula 02 de HTML e CSS

Links

Aqui, vamos aprender a fazer links internos, externos e de âncora.

O primeiro tipo de link é interno, ou seja, você navega dentro das páginas que estão nos seus arquivos do site, as que você mesmo criou, especificando o local onde elas estão: <a href="pages/nome.html"> . Lembrando que você deve localizar a pasta correta, colocando um ../ para voltar para o diretório anteriou ou simplesmente um / para voltar para sua pasta raiz da página

Caso você queira redirecionar o usuário para uma página que seja fora da sua (de outra pessoa), você ao invez de especificar o diretório (local), você indica o URL da página que quer: <a href="https://link.com/"> .

O link possui vários estados, podendo ser hover (mouse passando em cima), visited (depois de visitado) e active (quando você clica no link). Cada um deles pode ser individualmente alterado, podendo mudar qualquer propriedade do CSS com esses estados.

Você também pode criar um link com um arquivo para download, que é básicamente a mesma coisa do link interno, mas ao invés de apontar o local para um arquivo .html, você aponta para outro, como esse aqui.
