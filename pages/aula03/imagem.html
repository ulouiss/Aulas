Aula 03 de HTML e CSS

Imagem

Assim que se coloca uma imagem de plano de fundo num elemento.

Primeiro você usa o background-image: url(''); para definir uma imagem de acordo com um local ou um link. depois você pode usar o background-repeat: no-repeat; para que a imagem não se repita caso o tamanho seja menor que o do container. Para posicionar essa imagem você utiliza o background-position: center center;

1-) Qual a diferença entre o no-repeat, repeat-x e repeat-y?

R: o no-repeat não repete a imagem nenhuma vez independente da situação. o repeat-x repete a imagem apenas na horizontal e o repeat-y repete apenas na vertical
