Perguntas

1-) Qual a utilidade de trabalhar com o Github dentro do VScode?

R: Com ele é possivel fazer versionamento, que é útil principalmente para servir de backup, já que grava os arquivos em nuvem, que pode ser usado para voltar o sistema para versões passadas em caso de bug. No Github também você pode subir o site para o github pages que é uma especie de hospedagem gratuita para seu site.
